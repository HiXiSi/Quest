## 📝 开发备注

### 技术栈
- 前端：Vue 3 + Element Plus + Vite
- 后端：Go + Gin + GORM + SQLite
- 认证：JWT Token

### 启动命令
```bash
# 启动项目（推荐）
.\start.bat

# 或分别启动
cd backend && go run main.go
cd frontend && npm run dev
```
